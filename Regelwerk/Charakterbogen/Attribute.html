Attribute

Wird eine Aktion ausgeführt, die einem Attributswert entspricht (zB jemand möchte lügen), dann wird eine einfache Probe auf diesen Wert abgelegt. Soll eine spezifische Aktion ausgeführt werden, werden Proben auf drei Attributswerte verlangt (zB ein Hemd nähen würde Proben auf Umgang mit Werkzeugen, Fingerfertigkeit und Geduld verlangen). Gewisse häufige Aktionen sind bereits als Kombis festgelegt und bestehen aus lediglich zwei Attributen (zB erfordert ein Stehlen Proben auf Fingerfertigkeit und Schleichen).

  • Fokus+Beweglichkeit=Werfen
